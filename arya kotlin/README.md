# Arya Kotlin - Namma-Skill Android App

This is a native Kotlin Android version of the Namma-Skill project.

## Features

- Course discovery for skill development batches
- Search by course, trade, or center
- Trade and duration filters
- Nearby center list
- Success stories
- Candidate application form
- Candidate summary generation
- Local saved application history using SharedPreferences
- Copy summary and clear saved applications

## How to Run

1. Open the `arya kotlin` folder in Android Studio.
2. Let Android Studio sync Gradle.
3. Run the `app` configuration on an emulator or Android phone.

If Android Studio asks for SDK updates, install the recommended Android SDK and Gradle plugin components.
